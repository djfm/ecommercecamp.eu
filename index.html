How Does It Work??

The Cloud Platform (1/2)

Close partnership with OVH, Europe's leading hosting provider

Rock-solid technology stack already powering hundreds of thousands of websites worldwide (they have over 180,000 physical servers)

The Cloud Platform (2/2)

2 datacenters (Europe + Canada) for optimized response times

Multiple PHP and MySQL instances in each datacenter + daily backups => High availability + Fault tolerance

The Software (1/2)

Our Challenge

Maintain compatibility of existing add-ons and templates with our Cloud shops

Make the Cloud environment friendly to developers

Allow same level of customization in Cloud as in Download

Make the environment friendly to our DevOps :)

All of the above while staying true to our Open-Source philosophy!

The Software (2/2)

Our Solution

Use the same software in Cloud and Download

Shops are pre-generated to allow peaks of thousands of shop creations / day

Common files are shared between the shop instances (Core files)

Allow customization by providing FTP access to shop-specific files
